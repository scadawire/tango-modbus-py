# tango-modbus-py 

scadawire/tango-controls integration to Modbus RTU/TCP with Python DS 

# Structure

The integration makes use of the tango python binding to implement device driver functionality.
See also https://tango-controls.readthedocs.io/en/latest/development/device-api/python/index.html

The modbus specific functionality is covered by the pymodbus python package.
See also https://pypi.org/project/pymodbus/
